# AlgoViz v2.4 – Interactive Data Structures & Heaps Visualizer

AlgoViz is an interactive visualization tool for exploring tree-based data structures and heap operations. The application provides animated demonstrations of insertions, deletions, traversals, balancing operations, and memory representations to help users better understand how these structures work internally.

https://tree-and-heap-animator-808801576430.asia-southeast1.run.app/

## 📌 Overview

This project was created using a prompt-driven development workflow with Google AI Studio. Rather than manually programming the application, the project was built through iterative AI-assisted development, where requirements, features, user interface designs, bug reports, and refinements were provided as prompts and continuously improved through testing and feedback.

My primary role in this project was:

* Defining features and functionality
* Designing the user experience and interface behavior
* Creating and refining prompts
* Testing generated implementations
* Debugging issues through iterative prompt engineering
* Validating algorithm visualizations and application behavior

The goal was to explore how modern AI-assisted development can be used to create a complex educational application with minimal manual coding.

## ✨ Features

### Tree Visualizations

Supported tree structures include:

* Binary Search Tree (BST)
* AVL Tree
* Splay Tree
* B-Tree (Order 3)
* B+ Tree (Order 3)

Users can perform common operations and observe how the structure changes through step-by-step animations.

### Heap Visualizations

Supported heap structures include:

* Min Heap
* Max Heap
* Fibonacci Heap

Operations are animated to demonstrate heap property maintenance and element movement.

### Traversal Visualization

The application displays:

* Preorder Traversal
* Inorder Traversal
* Postorder Traversal

Traversal paths are highlighted in real time as algorithms execute.

### Algorithm Tracking

A synchronized code-tracking panel highlights the currently executing algorithm step during animations, helping users connect visual changes to algorithm logic.

### Memory Representation

The visualizer includes an array-based memory view that demonstrates how hierarchical structures can be represented in sequential memory layouts.
