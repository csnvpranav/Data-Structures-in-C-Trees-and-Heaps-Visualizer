## 📌 Overview

AlgoViz is an interactive visualization tool that demonstrates the behavior of various tree-based data structures and heaps through real-time animations. The project helps users understand how operations such as insertion, deletion, traversal, balancing, and heapification affect the internal structure of these data structures.

This project was built using a prompt-driven development workflow with Google AI Studio. Instead of manually writing the implementation code, the application was created through iterative prompting, testing, debugging, and refinement of AI-generated code.

My contribution focused on:

- Defining project requirements
- Designing the user interface and user experience
- Creating and refining prompts
- Testing generated implementations
- Identifying and fixing bugs through iterative prompting
- Validating algorithm behavior and visualizations

The project serves as an experiment in AI-assisted software development while also functioning as an educational visualization platform.

## ✨ Features

### Supported Tree Structures

- Binary Search Tree (BST)
- AVL Tree
- Splay Tree
- B-Tree (Order 3)
- B+ Tree (Order 3)

### Supported Heap Structures

- Min Heap
- Max Heap
- Fibonacci Heap

### Visualization Features

- Step-by-step operation animations
- Tree balancing visualization
- Traversal path highlighting
- Preorder, Inorder, and Postorder traversal outputs
- Memory representation using array layouts
- Real-time algorithm tracking panel
- Responsive desktop and mobile interface
- Dark Mode and Light Mode support
